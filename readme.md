## Requirements

- [Node.js v10+](https://nodejs.org/) + [Yarn](https://yarnpkg.com/)

- [Expo CLI](https://docs.expo.io/workflow/expo-cli/)


## Usage

```sh
# install dependencies
npm install

# run bundler
expo start

# run on Android device/emulator
npm run android

# run on iOS device/simulator
npm run ios

# run tests
npm run test

```
